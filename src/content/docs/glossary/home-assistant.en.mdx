## What is Home Assistant?

**Home Assistant** is a free, open-source home automation platform designed to run on local hardware. It serves as a central hub for controlling and automating smart devices from hundreds of different manufacturers.

## Key Features

- **Local control**: Runs on your own hardware — no cloud dependency
- **2,000+ integrations**: Supports Zigbee, Z-Wave, Wi-Fi, Bluetooth, and cloud services
- **Powerful automations**: Visual editor and YAML-based automation engine
- **Dashboards**: Customizable Lovelace UI for monitoring and control
- **Add-ons**: Extend functionality with community-built add-ons
- **Privacy-focused**: Your data stays on your network

## Installation Options

- **Home Assistant OS** — Dedicated OS for Raspberry Pi or x86 hardware
- **Home Assistant Container** — Docker container
- **Home Assistant Supervised** — Managed install on Debian
- **Home Assistant Core** — Python virtual environment
