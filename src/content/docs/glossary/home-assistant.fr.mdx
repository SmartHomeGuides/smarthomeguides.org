## Qu'est-ce que Home Assistant ?

**Home Assistant** est une plateforme domotique gratuite et open source conçue pour fonctionner sur votre propre matériel. Elle sert de hub central pour contrôler et automatiser des appareils connectés de centaines de fabricants différents.

## Caractéristiques principales

- **Contrôle local** : Fonctionne sur votre matériel — aucune dépendance au cloud
- **Plus de 2 000 intégrations** : Supporte Zigbee, Z-Wave, Wi-Fi, Bluetooth et les services cloud
- **Automatisations puissantes** : Éditeur visuel et moteur d'automatisation YAML
- **Tableaux de bord** : Interface Lovelace personnalisable pour le suivi et le contrôle
- **Extensions** : Fonctionnalités étendues grâce aux add-ons communautaires
- **Respect de la vie privée** : Vos données restent sur votre réseau

## Options d'installation

- **Home Assistant OS** — Système dédié pour Raspberry Pi ou matériel x86
- **Home Assistant Container** — Conteneur Docker
- **Home Assistant Supervised** — Installation gérée sur Debian
- **Home Assistant Core** — Environnement virtuel Python
